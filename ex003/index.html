Operadores e seus tipos

Aritméticos

Eles são +, -, *, /, %, **

  • O + indica soma
  • o - indica subtração
  • O * indica vezes
  • O / indica divisão
  • O % indica divisão pelo resto
  • O ** indica elevação

Atribuições

Eles fazem atribuições para as variaveis exemplos

  • var a = 5 + 3
  • var b = a % 5
  • var c = 5 * b ** 2
  • var d = 10 - a / 2
  • var e = 6* 2 / d
  • var f = b % e + 4 / e
